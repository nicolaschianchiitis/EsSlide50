# EsSlide50
Esercizio slide 50 (Node.JS)

# Consegna su Classroom
Utilizzare i Tag HTML5 relativi ai campi del form per costruire il seguente modello
da validare (controllare se i dati dell’input del form sono congrui all’applicazione)
sul server web in Node.JS (usa il metodo di invio get):

- Nome
- Cognome
- Età
- Sesso: m,f
- Stato Civile: celibe/nubile, coniugato/a, vedovo/a, divorziato/a
- Hobby preferito: tra più scelte predefinite.

Nella prima versione del programma il controllo di validazione compiuto dal server
restituisce un messaggio che evidenzia gli errori di compilazione compiuti
dall’utente. Nella seconda versione il controllo di validazione restituisce al
client una pagina html con il form che presenta i campi compilati in modo corretto
e vuoti i campi che non sono congrui all’elaborazione del server (ad esempio quelli vuoti).
Infine aggiungere al form un campo per l'upload di un file (es. una foto del profilo).
